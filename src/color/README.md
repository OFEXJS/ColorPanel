# 颜色模块工具介绍

本模块提供了全面的颜色处理解决方案，支持多种色彩模式、交互操作及实用工具，适用于前端开发、设计工具等场景。

## 🌈 主要功能

### 1. 多色彩模式支持

支持六种主流颜色表示方式：

- **HEX**：十六进制颜色代码（如 `#FF5733`）
- **RGB**：红绿蓝三原色模式（如 `rgb(255, 87, 51)`）
- **RGBA**：带透明通道的 RGB（如 `rgba(255, 87, 51, 0.8)`）
- **HSL**：色相、饱和度、亮度模式（如 `hsl(9, 100%, 60%)`）
- **HSB**：色相、饱和度、明度模式（如 `hsb(9, 100%, 100%)`）
- **CMYK**：印刷四色模式（如 `cmyk(0%, 66%, 80%, 0%)`）

### 2. 交互功能

- 🎨 点击色块打开颜色选择面板
- 📁 手风琴式预设颜色展示，收纳与展开自如
- 📋 一键复制任意格式的颜色值
- 🔄 实时颜色格式转换，所见即所得
- 🖱️ 拖拽式颜色选择器，直观调色
- 🎚️ 透明度滑块调节，轻松控制 Alpha 通道

### 3. 响应式设计

- 📱 完整移动端适配
- 🌙 暗色模式自动支持
- 👆 触摸交互优化，触屏友好
- ♿ 无障碍访问支持，符合 WAI-ARIA 标准

## 🧩 核心组件

- **ColorPicker**：高级颜色选择器，支持多种颜色模型与直观操作
- **ColorPanel**：综合颜色管理面板，集成预览、调节与保存功能
- **ColorConverter**：实时颜色格式转换器，支持双向转换
- **ColorInput**：智能颜色输入框，自动识别并验证多种格式
- **ColorModuleDemo**：完整功能演示组件，快速上手体验

## ⚙️ 工具函数

提供一系列开箱即用的颜色处理函数：

- **格式转换**：`hexToRgb`、`rgbToHsl`、`rgbToCmyk` 等
- **颜色分析**：`isDarkColor`、`getContrastTextColor`
- **颜色操作**：`adjustBrightness`、`calculateColorDifference`
- **实用功能**：`copyToClipboard`、`generateRandomColor`
